CSCI 401 Operating Systems
Project 1, Part 1 - A Linked List in the C programming langauge.

Author: Fleurevca Francois
Description: This project requests that a C-string linked list data structure for storing character strings is created.
It includes functions that will enable you to create a list, add items to a list, removes item from the list, print the list,
flush or free the list. 

Compiling and Running the Program:
- In a Linux or Mac based system run "make" in the main directory.
- In a Windows, a linux bassed subsystem is needs to be downloaded and once that is up and running, running the command "make" 
  in the main directory.

Challenges:
- Learning the C langauge and learning to create a Makefile
